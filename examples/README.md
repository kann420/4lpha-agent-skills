# Examples

Current home for:

- generated strategy-spec outputs
- market-context snapshots
- judge-friendly walkthrough artifacts

Generated files are written by:

```powershell
npm run demo
npm run cli -- strategy generate
npm run generate:cmc-strategy
```

The judge-friendly bundle now includes:

- `cmc-market-context.snapshot.json`
- `fourmeme-discovery.snapshot.json`
- `cmc-market-regime.strategy.json`
- `demo.summary.md`
