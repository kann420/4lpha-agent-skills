# Hackathon Brief

## Working Interpretation

This repo is aimed at the BNB hackathon `Strategy Skills` track.

The project thesis is:

- use broad market context from CoinMarketCap / Agent Hub
- apply Four.Meme-specific filters for BNB meme-token setups
- output an explicit strategy spec that can be backtested

## Why The Four.Meme Angle Works

Four.Meme is not a mismatch if we frame it correctly.

It helps because:

- it gives the project a clear BNB-native niche
- it differentiates the repo from generic market-commentary skills
- it lets us reuse durable knowledge already developed in `4alpha`

It fails only if we reduce the project to:

- a venue-specific trade executor
- a wallet wrapper
- a ticker-picking prompt without explicit rules

## Desired Prize Alignment

- Main track: Strategy Skills
- Special prize: Best Use of Agent Hub
- Special prize: Best Use of BNB AI Agent SDK
